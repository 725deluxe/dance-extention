# DancePractice — ダンス練習プレイヤー

YouTubeでダンスを練習するためのChrome拡張機能です。  
動画ページを開くと右下にパネルが表示され、**左右反転・ABリピート・速度変更**が使えます。

---

## こんな人に

- YouTubeの振り付け動画を見ながらダンスを練習している
- 鏡越しに練習したいので動画を左右反転したい
- 難しい箇所だけ何度も繰り返して練習したい
- スローモーションで動きを確認したい

---

## 機能

| 機能 | 内容 |
|---|---|
| **左右反転** | 動画をワンクリックで鏡像表示に切り替え |
| **速度変更** | 0.25× / 0.5× / 0.75× / 1× / 1.25× / 1.5× |
| **ABリピート** | A点・B点を指定した区間だけ無限ループ |
| **全体ループ** | AB未設定のときは動画全体をループ |
| **ループ回数** | 現在の動画のループ回数を表示 |
| **最小化** | パネルを折りたたんで動画を広く使える |

---

## インストール方法

ChromeのWeb Storeには未公開のため、以下の手順で手動インストールします。  
3分でできます。

**1. ファイルをダウンロード**

このページ右上の緑のボタン「**Code**」→「**Download ZIP**」をクリック

**2. ZIPを解凍**

ダウンロードした `dance-practice-extension-main.zip` を解凍します。

**3. Chromeに読み込む**

1. Chromeのアドレスバーに `chrome://extensions` と入力してEnter
2. 右上の「**デベロッパーモード**」をオンにする
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
4. 解凍してできたフォルダの中の `dance-extension` フォルダを選択

**4. 完了**

YouTubeの動画ページを開くと、右下にパネルが表示されます。

---

## 使い方

**左右反転：**

「左右反転 OFF」ボタンをクリックするとONになり、動画が鏡像表示になります。  
もう一度クリックで解除。

**ABリピートの手順：**

1. 動画を再生し、繰り返したい区間の**始まりで「A点」**をクリック
2. 終わりで**「B点」**をクリック
3. 自動的にA→Bをループし続けます
4. 「クリア」で解除（動画全体のループに戻ります）

**速度変更：**

0.25×のスローモーションから確認して、慣れてきたら1×に戻す使い方がおすすめです。

**最小化：**

パネル右上の「—」ボタンで折りたたみ、「＋」で展開できます。
